## 1. 工资表 (salary_table)

### 表说明
存储员工工资详细信息，包含基础信息、出勤信息、薪资构成、扣款明细等30个字段。

### 字段详细说明

#### 主键字段

| 字段名 | 数据类型 | 约束 | 说明 |
|--------|----------|------|------|
| id | INTEGER | PRIMARY KEY AUTOINCREMENT | 自增主键 |

#### 基础信息类（文本类型）

| 字段名 | 数据类型 | 约束 | 说明 | 示例值 |
|--------|----------|------|------|--------|
| 序号 | TEXT | NOT NULL | 员工编号 | "001" |
| 部门 | TEXT | NOT NULL | 所属部门 | "护理部" |
| 姓名 | TEXT | NOT NULL, UNIQUE | 员工姓名（唯一） | "张三" |
| 岗位 | TEXT | - | 岗位名称 | "护理员" |

#### 日期类

| 字段名 | 数据类型 | 约束 | 说明 | 格式 | 示例值 |
|--------|----------|------|------|------|--------|
| 入职日期 | DATE | NOT NULL | 入职日期 | YYYY-MM-DD | "2023-01-15" |

#### 出勤类（数字类型）

| 字段名 | 数据类型 | 约束 | 说明 | 精度 | 示例值 |

| 应出勤天数 | DECIMAL(4,1) | >= 0 | 应出勤天数 | 1位小数 | 22.0 |
| 实际出勤天数 | DECIMAL(4,1) | >= 0, <= 应出勤天数 | 实际出勤天数 | 1位小数 | 21.5 |
| 上门服务小时 | INTEGER | >= 0 | 上门服务小时数 | 整数 | 15 |

| 应发工资 | DECIMAL(10,2) | >= 0 | 应发工资 | 2位小数 | 5300.00 |

| 字段名 | 数据类型 | 约束 | 说明 |
|--------|----------|------|------|
| id | INTEGER | PRIMARY KEY AUTOINCREMENT | 自增主键 |

#### 基础信息类（与工资表1:1映射）

| 字段名 | 数据类型 | 约束 | 说明 | 映射关系 |

## 3. 用户表 (users)

### 表说明
用户身份认证与权限管理，基于RBAC权限模型。

### 字段详细说明

| 字段名 | 数据类型 | 约束 | 说明 | 示例值 |
|--------|----------|------|------|--------|
| id | INTEGER | PRIMARY KEY AUTOINCREMENT | 自增主键 | - |
| 姓名 | TEXT | NOT NULL, UNIQUE | 用户姓名（唯一） | "张三" |
| role | TEXT | CHECK IN ('user', 'admin') | 用户角色 | user/admin |
| token | TEXT | - | 登录令牌 | JWT字符串 |
| token_expire_time | TIMESTAMP | - | Token过期时间 | 2024-01-02 10:00:00 |
| last_login_time | TIMESTAMP | - | 最后登录时间 | 2024-01-01 10:00:00 |
| created_at | TIMESTAMP | DEFAULT CURRENT_TIMESTAMP | 创建时间 | - |

### 索引

| 索引名 | 字段 | 说明 |
|--------|------|------|
| idx_users_name | 姓名 | 按姓名快速查询 |
| idx_users_token | token | 按Token快速验证 |

## 4. 签名文件表 (signature_files)

### 表说明

## 📋 数据格式规范总结

### 金额类数据
- **数据类型**：DECIMAL(10,2)
- **精度**：保留2位小数
- **约束**：>= 0
- **示例**：3000.00

### 出勤类数据
- **数据类型**：DECIMAL(4,1)
- **精度**：保留1位小数
- **约束**：>= 0, 实际出勤天数 <= 应出勤天数
- **示例**：22.0

### 上门服务小时
- **数据类型**：INTEGER
- **精度**：整数
- **约束**：>= 0
- **示例**：15

### 日期类数据
- **数据类型**：DATE
- **格式**：YYYY-MM-DD
- **示例**：2023-01-15

### 时间戳类数据
- **数据类型**：TIMESTAMP
- **格式**：YYYY-MM-DD HH:MM:SS
- **示例**：2024-01-01 12:00:00

### 布尔类数据
- **数据类型**：BOOLEAN
- **取值**：true/false
- **示例**：true

## 🔒 数据完整性约束

### 主键约束
- 所有表都有自增主键 `id`

### 唯一约束
- 工资表.姓名：UNIQUE
- 用户表.姓名：UNIQUE
- 签名文件表.文件ID：UNIQUE

### 外键约束
- 签名文件表.用户姓名 → 用户表.姓名
- 操作日志表.操作人 → 用户表.姓名

### 检查约束
- 金额类：>= 0
- 出勤类：实际出勤天数 <= 应出勤天数
- 文件大小：<= 10M (10485760字节)
- 查询次数：>= 0
